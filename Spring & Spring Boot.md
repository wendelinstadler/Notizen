# Spring & Spring Boot

## Spring

Spring ist ein Java-Framework, das die Entwicklung von Enterprise-Java-Anwendungen vereinfacht und die Strukturierung des Codes unterstützt. Es bietet zahlreiche Module für verschiedene Anforderungen, wie:

- Datenbankzugriff
- Sicherheit
- Webentwicklung
- und mehr

Spring fördert die **lose Kopplung** von Komponenten und ermöglicht die Implementierung von **Inversion of Control (IoC)** sowie **Dependency Injection (DI)**.

---

## Spring Boot

Spring Boot ist eine Erweiterung des Spring-Frameworks und zielt darauf ab, den Entwicklungsprozess weiter zu vereinfachen. Es bietet:

- eine "Starter"-Infrastruktur mit minimaler Konfiguration
- schnelle Entwicklung von Produktionsanwendungen
- automatische Konfiguration und weniger Boilerplate-Code (wiederholender Code)
